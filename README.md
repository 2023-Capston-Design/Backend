## Run with native Node.js

1. Install packages
```
yarn install
```

2. Run API server
```
// Development mode
yarn run start:dev

// Production mode
yarn run start
```

3. Open API Document
```
localhost:3000/docs
```

## Dev-Docker

```
yarn install

cd dev-docker

docker-compose up -d
```

## Production Docker

```
cd production-docker

docker-compose up -d
```
